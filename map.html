UNESCO Map and Legend
This is just a better look at the map and legend design for the UNESCO project. I wanted to keep the color of the map in theme with the rest of the project which is why the map features the orange and blue color scheme. I originally wanted to make it a topographical map to show off the mountainous region but opted for elevation map with different shades of orange to represent the changes in elevation. I think that this is one of the stronger parts of the UNESCO project which is why it pops up in many other aspects of this project.
